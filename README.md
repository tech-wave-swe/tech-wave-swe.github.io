## How to run

Requisiti:
- Node.js
- TypeScript compiler (tsc)
- Visual Studio Code

### Steps

1) Clonare questa repo localmente

2) Eseguire il seguente comando
	```bash
	npm install
	```
3) Premere F5 o andare sulla sezione "Run and Debug" ed eseguire la task "Run Extension"
